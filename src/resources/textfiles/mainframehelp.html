Wave intensity analysis options

There are various ways that wave analysis can be performed. These are:

Separate flow and pressure sources (i.e. Doppler FloWire)

  • Select this option if your pressure and flow data are stored in two separate files (i.e. recorded from two separate wires) You will be prompted to upload two distinct data files: one for the pressure readings and one for the flow velocity readings. The application will then guide you through the process of performing Wave Intensity Analysis (WIA) by combining these two sources.

Same flow and pressure sources (i.e. ComboWire)

  • Select this option if your data was recorded concurrently and is in the same file (i.e. single combination wire, ComboWire). You will be prompted to upload a single data file that contains columns for both pressure and flow data.

Re-analyze prior wave profile

  • Choose this option to load a previously saved WIA analysis file (a .wia file). This allows you to review, modify, or re-run the analysis without having to re-upload the original raw data. This is useful for adjusting wave selections (or adding new waves), alignment, or analysis parameters.
